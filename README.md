## Певые шаги
1. Скачайте архив (zip) со страницы [последнего релиза](https://github.com/FunsyMe/Ninja-Service/releases/latest)
2. Распакуйте содержимое архива по пути, который не содержит кириллицу или пробелы
3. Запустите файл `ninja_service.bat` (`ninja_service`)
4. Выберите пункт 9 (Запустить Авто-Поиск конфига)
5. Ожидай, пока тест не выведет тебе подходящий конфиг
6. Обязательно запомни конфиг. Он понадобится в будущем

## Использование
1. Запусти нужный конфиг из папки `pre-configs`
2. Поздравляю! Discord, YouTube и другие сервисы работют

## Установка в Авто-Запуск
1. Запустите файл `ninja_service.bat` (`ninja_service`)
2. Выбери пункт 1 (Установить Zapret в Авто-Запуск)
3. Выбери свой кофниг (НЕ номер кофнига в его имени)
4. Ожидай, пока Zapret не установится в Авто-Запуск
5. Поздравляю! Discord, YouTube и другие сервисы работют
